Interface Documentation

Window Name
Main Page
Description
Main page of website, contains welcome message and links to different subsites with paintings and drafts descriptions.
Sections
Header Section with page name, navigation links and translation buttons.
Welcome Message Section with a short welcome message.
Paintings Section with items containing paintings picture, name and link to subsite with more information.
Footer Section with version and copyrights information.
Actions
Home Link to the welcome message on main page.
About me Link to the page with information about website owner.
English Translation Button Translates the webpage to English language.
Polish Translation Button Translates the webpage to Polish language.
German Translation Button Translates the webpage to German language.
View More Loads the subsite of chosen painting.
