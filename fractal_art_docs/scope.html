Project Scope

Project Description
The aim of this project is to create a website to showcase the art created with the use of fractals. User can find there information about various paintings, owner and link to social media.
Problem Benefits of Solving
User should see the list of paintings. Implementation of a list of paintings on main page will allow user to get an overview of the website content.
User should be able to see paintings details. Each painting described and showcased on it's own subsite will allow user to get more information.
Section about user should be accessible. A section with description and links to relevant social platforms will allow user to get more information about artist.
Indicative Schedule
Implementation Start Date: 05.08.2024 Implementation End Date: 12.08.2024
Solution Scope
During implementation the developer will:
  • Design the website.
  • Implement the webpages using html with css.
  • Implement the JavaScript code for translations.
  • Prepare the Polish and German translations.
  • Deploy the website on GitHub Pages.
Project Assumptions
  1. Deplyment done by GitHub Pages.
  2. Usage of file generated by Python script as website background.
  3. Continous development and maintenance after the end date in indicative schedule.
Key Risks Rating
Error on GitHub pages preventing users from visiting page. Low
Page growing too big over time to be deployed on GitHub pages. Medium
