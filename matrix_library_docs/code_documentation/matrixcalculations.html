Class: MatrixOperations

Description: The MatrixOperations class provides various methods for performing matrix calculations such as addition, subtraction, multiplication, and inversion.

Methods

Method Name Description
addMatrix(matrixA, matrixB) Performs addition of two matrices and returns the result as a new matrix.
subtractMatrix(matrixA, matrixB) Performs subtraction of matrixB from matrixA and returns the result.
multiplyMatrix(matrixA, matrixB) Multiplies two matrices and returns the product as a new matrix.
invertMatrix(matrix) Returns the inverse of the given matrix if it exists.
transposeMatrix(matrix) Transposes the given matrix and returns the transposed matrix.
determinant(matrix) Calculates the determinant of the matrix.
isSquareMatrix(matrix) Checks if the given matrix is a square matrix (equal number of rows and columns).
