Class: MatrixIO

Description: The MatrixIO class provides methods for operations on files. It allows to save and load a matrix.

Methods

Method Name Description
load_matrix(path) Method loads the matrix from file in given path.
save_matrix(path, vect) Method saves given matrix to file in given path.
