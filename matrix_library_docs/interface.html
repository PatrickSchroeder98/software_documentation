Interface Documentation

Window Name
Microsoft Visual Studio Debug Console
Description
This application does not have a graphical user interface. The methods in the library can be integrated with other programs, providing matrix calculation functionality. The changes made by the methods can be viewed using a console, such as the Microsoft Visual Studio Debug Console. The MatrixCalculationsAPI class is designed as a console interface for easy use of the application by the end user.
