Window Name
Project Main Page
Description
Main page of project documentation, contains title and index with links to documentation pages.
Sections
Header Section with webpage name and navigation links.
Technology Logo Logo of the main technology used in the project.
Project Title Full title of the project.
Index Index of all project documentation pages.
Next Link to next page of documentation.
Additional Media Link to any additional project media.
Footer Section with copyrights information.
Actions
Home Link to the greeting message on main page.
Project Link to the project main page.
Project Pages Links Links to the project documentation pages, such as scope or requirements.
Next Link to the next project documentation page listed in index. Project main page doesn't have 'Previous' link.
Project Media Link Link to the project media, such as code documentation or website link.
